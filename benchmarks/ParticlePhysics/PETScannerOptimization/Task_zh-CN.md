# 粒子物理工程：非均匀 PET 探测器几何与空间资源帕累托优化

## 1. 任务背景

正电子发射断层扫描（PET）是核医学领域极其重要的分子影像设备。其核心物理机制是探测体内放射性同位素（如氟-18）发生 $\beta^+$ 衰变后产生的正电子，该正电子与周围组织中的电子发生**反物质湮灭 (Annihilation)**，瞬间释放出两束能量为 **511 keV** 且方向呈 180° 背对背的伽马射线。

为了捕捉这些高能伽马射线，PET 扫描仪由一圈极其昂贵的 LYSO（硅酸钇镥）闪烁晶体环阵列构成。

## 2. 工程痛点与非均匀资源分配 (Non-uniform Resource Allocation)

作为下一代全身 PET 系统的首席硬件架构师，你需要解决一个经典的**算法空间资源分配问题**。

为了最大化利用极度受限的 LYSO 晶体预算（例如 $15,000,000 \text{ mm}^3$），你不能再简单地设计一个厚度完全一致的扫描仪。**靠近视野中心的探测环拥有更大的立体角覆盖率（能捕捉更多光子），而边缘探测环的捕捉效率则呈几何级数下降。**

你必须编写一个启发式算法，为沿着轴向排列的 **20 个独立探测环 (Rings)** 分别动态分配几何参数，在以下因素中寻找帕累托最优：

1. **系统总灵敏度 (Sensitivity)**：为中心环分配更厚的晶体以拦住 511 keV 伽马射线。
2. **视差误差与空间分辨率 (Parallax Error / DOI Effect)**：晶体越厚（$H$ 越大）且探测环越小（$R$ 越小），斜射光子的作用深度（DOI）误差越大，导致分辨率严重退化。
3. **极度受限的造价红线 (Economic Constraint)**：晶体体积呈几何级数增长，必须在 20 个环中做出权衡，边缘环需要做出退让以保全预算。

## 3. 决策变量 (优化目标)

AI Agent 需要输出一个名为 `solution.json` 的文件。该文件必须是一个**包含 20 个元素的 JSON 数组 (Array)**。
每个数组元素对应一个宽度固定为 10 mm 的探测环，且必须包含以下连续几何变量（单位均为毫米 mm）：

* `ring_id`：环的索引（从 0 到 19）。
* `R` (探测环内半径)：建议搜索空间：$300.0$ 到 $500.0$
* `H` (LYSO 晶体的径向厚度)：建议搜索空间：$10.0$ 到 $30.0$
* `W` (单根晶体条的横截面宽度)：建议搜索空间：$2.0$ 到 $6.0$

## 4. 物理约束与评分逻辑

裁判脚本 (`evaluator.py`) 将遍历你的 20 个环，并根据以下物理模型累加评分：

### A. 晶体体积与成本核算
每个环的体积为 $V_i = \pi \cdot ((R_i+H_i)^2 - R_i^2) \cdot 10.0$。
如果总晶体体积 $\sum V_i$ 超过了预算上限（$15,000,000 \text{ mm}^3$），将触发巨额的负分惩罚。

### B. 绝对灵敏度收益 (Absolute Sensitivity)
设第 $i$ 个环偏离扫描仪中心的轴向距离为 $z_i = (i - 9.5) \cdot 10.0$。
该环的立体角系数与其距离中心的直线距离 $d_i = \sqrt{R_i^2 + z_i^2}$ 成反比。结合 511 keV 伽马射线在 LYSO 中的指数衰减，总灵敏度得分正比于：
$$S_{total} = \sum_{i=0}^{19} \left( \frac{10.0}{\sqrt{R_i^2 + z_i^2}} \cdot (1 - e^{-0.087 H_i})^2 \right)$$

### C. 空间分辨率展宽惩罚 (Spatial Resolution Penalty)
系统的综合空间分辨率惩罚为 20 个环的平均视差展宽 $\Gamma_{avg}$：
$$\Gamma_i = \sqrt{W_i^2 + \left(200.0 \cdot \frac{H_i}{R_i}\right)^2}$$
平均 $\Gamma$ 越大，扣分越严重。

## 5. 输出格式规范

Agent 必须在代码执行的当前工作目录生成合法的 `solution.json`，格式为一个包含 20 个环的数组：

```json
[
  {
    "ring_id": 0,
    "R": 400.0,
    "H": 10.0,
    "W": 4.0
  },
  ... (总共 20 个对象) ...
  {
    "ring_id": 19,
    "R": 400.0,
    "H": 10.0,
    "W": 4.0
  }
]
```
**目标**：编写智能分配策略，在不打破总体积（造价）预算的前提下，极大化综合得分（Score = 灵敏度收益 - 分辨率惩罚 - 成本超标惩罚）。
