# 粒子物理与核技术工程 (Particle Physics and Nuclear Engineering)

[English](./README.md) | 简体中文

## 1. 领域背景
粒子物理与核技术涉及对基本粒子及其相互作用的研究。在现代大型科学工程（如高能粒子对撞机、宇宙射线探测、核辐射监测）中，探测器阵列的布局、数据筛选阈值的设定以及加速器参数的控制，往往面临极其复杂的物理约束和高昂的经济成本。

引入 AI Agent 解决该领域的开放在线优化问题，能够在保证物理目标（如统计显著性、信噪比）的前提下，极大降低工程造价和试错成本。

## 2. 任务索引 (Sub-task Index)

目前本领域包含以下基准测试任务：

* **[缪子成像探测器阵列布局优化 (Muon Tomography Optimization)](./MuonTomography/README_zh-CN.md)**
  * **背景**：利用宇宙射线缪子对大型结构进行内部透射成像。
  * **目标**：在预算受限且需考虑挖掘成本的情况下，寻找最佳的探测器空间坐标与角度布局，最大化目标区域的有效接收通量。
